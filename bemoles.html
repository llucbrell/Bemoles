Automatic html canvas innsert with Midi

Using a simple JavaScript code bemoles can generate sheet music from an standard midi file.

If you click on the sheet music you can listen it.

To create it put an HTML tag with some id. Then you can load the sheet passing the path to the midi file, your tag id and an index as a midi array controler.

For non-developers users only count your midi files, and assign a number for each one in order of appearance in your website. Remember, starting with 0.

Visual test for figure recognition

JavaScript Code

midiBemoles('prueba4.mid', 'testnotas', 0);

'prueba4.mid'--> it's the path

'testnotas'--> it's the div id

0 --> its the first midi file
